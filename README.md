# わたしの日本酒図鑑

## サービス概要
旅先や居酒屋で飲んだ日本酒を登録し自分だけの日本酒リストを作れるサービスです。

## このサービスへの思い・作りたい理由
今まで居酒屋や旅先などで、たくさんの日本酒を飲んできました。
その時は「これ美味しい！また飲みたい！」と思っても、いざ同じ銘柄に出会ったときに
「あれ？これだったっけ？」と迷うことがよくあります。
日本酒は同じ銘柄でも、純米大吟醸や吟醸など製法の違いで味が大きく変わることもあり、
「同じ名前でもこんな味だったかな？」と感じることもあります。
また人におすすめしたいと思っても、「こんな味だった」とは覚えているのに、銘柄名が思い出せないこともありました。
そうした経験から飲んだ日本酒を忘れずもう一度出会えた時に確かに味に再会できるような”自分だけの日本酒記録帳”を作りたいと思いました。

## ユーザー層について
- 日本酒が好きな人
  - 日本酒に興味が出てきた社会人
  - 旅先や居酒屋で「美味しい」と思うけど名前を覚えられない

## サービスの利用イメージ
- 旅先や居酒屋で日本酒を飲んだ際に、アプリで日本酒の情報を登録することができます。
- 飲んだお酒を登録していくことで自分だけの銘柄リストを作ることができます。

## ユーザーの獲得について
- 日本酒が好きな友人に利用してもらう
- SNSやRUNTEQコミュニティでの共有

## サービスの差別化ポイント・推しポイント
Chromeで「日本酒記録アプリ」と検索した際に上位に出てくる「さけのわ」等のアプリは、  
記録できる機能の他にコミュニティ機能を持つものが多くあります。  
「わたしの日本酒図鑑」では、コミュニティ機能は持たせません。  
自分の記録を公表しないことによって、自分にとってはあまり美味しくなかった日本酒の感想等も  気軽に残すことができます。  
その結果、他人の評価やトレンドに左右されない自分の好みに正直な記録が増え、  
その記録を見直すことで自分の好みを深掘りすることができるようになり、  
その人にとって役立つ自分専用の日本酒リストを作ることができます。  
また、地域分布マップで飲んだ日本酒の産地が日本地図上で色づいていくことで、  
アプリの中で"日本酒をめぐる旅"を楽しむことができるのがこのアプリの推しポイントです。

## 機能候補
- MVPリリース時
  - TOPページ(概要等)
  - ユーザ登録・ログイン
  - 日本酒登録・編集・削除
  - 一覧・詳細表示
- 本リリース時
  - 検索機能
  - ソート機能
  - 日本酒チャート・可視化
    - 地域分布マップ(マップチャート)
    - 蔵元・銘柄分析チャート
  - バッジ機能
  - 飲みたいリスト
  - レスポンシブ対応

## 使用する技術スタック
- 使用するフレームワーク（Ruby on Rails7.2.2）
- データベース（PostgreSQL）
- デプロイ先（Render）
- 使用予定のGem
  - gem devise -ユーザ登録・ログイン
  - gem ransack -検索機能
  - gem rails-i18n -日本語対応 
  - gem kaminari -ページネーション
  - gem tailwindcss-rails - UI作成
